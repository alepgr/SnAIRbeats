\doxysubsection{Detailed Description}
Configuration settings for a gyroscope sensor. 

This structure defines the configuration parameters for a gyroscope sensor, including the sample rate, full-\/scale range, and Digital Low-\/\+Pass Filter (DLPF) settings.

Members\+:
\begin{DoxyItemize}
\item uint8\+\_\+t sample\+\_\+rate\+\_\+div\+: The sample rate divider, which determines the inner sample data rate of the gyroscope. The value should be in the range \mbox{[}0, 250\mbox{]}. The formula for calculating the inner sample rate is\+: 1.\+1k\+Hz / sample\+\_\+rate\+\_\+div
\item \doxylink{namespaceicm20948_ab4ed127a0673926c94b3419379774950}{gyro\+\_\+scale} scale\+: The full-\/scale range of the gyroscope measurements. This defines the maximum angular velocity (in degrees per second) that the sensor can measure. The available options are defined in the {\ttfamily \doxylink{namespaceicm20948_ab4ed127a0673926c94b3419379774950}{gyro\+\_\+scale}} enumeration.
\item bool dlpf\+\_\+enable\+: A flag indicating whether the Digital Low-\/\+Pass Filter (DLPF) is enabled. If true, the DLPF is active and the cutoff frequency is determined by {\ttfamily dlpf\+\_\+config}.
\item \doxylink{namespaceicm20948_ab21febc613fcda495faedf1fadf8da01}{gyro\+\_\+dlpf\+\_\+config} dlpf\+\_\+config\+: The configuration settings for the DLPF, determining the filter\textquotesingle{}s cutoff frequency. The available options are defined in the {\ttfamily \doxylink{namespaceicm20948_ab21febc613fcda495faedf1fadf8da01}{gyro\+\_\+dlpf\+\_\+config}} enumeration.
\end{DoxyItemize}
