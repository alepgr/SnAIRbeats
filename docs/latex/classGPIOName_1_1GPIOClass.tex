Event driven worker reading data when HIGH seen on GPIO. 

This function is an event driven interrupt controlled by a GPIO pin. void GPIOName\+::\+GPIOClass\+::\+Worker\+Data\+Collect (\begin{DoxyParamCaption}{}{}\end{DoxyParamCaption})}



Event driven worker reading data when HIGH seen on GPIO and records data to a CSV. 

The function begins by initialising a csv file named by the user. This function then uses blocking interrupts controlled by a GPIO pin. Once this GPIO pin reads HIGH the function reads the data registers using the Read\+Accel() callback from the IMU\textquotesingle{}s driver and appends this data into the opened CSV file.
