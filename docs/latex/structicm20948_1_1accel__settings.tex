\doxysubsection{Detailed Description}
Configuration settings for an accelerometer sensor. 

This structure defines the configuration parameters for the accelerometer, including the sample rate, full-\/scale range, and digital low-\/pass filter (DLPF) settings.

Members\+:
\begin{DoxyItemize}
\item uint16\+\_\+t {\ttfamily sample\+\_\+rate\+\_\+div}\+: The sample rate divider, which determines the inner sample data rate of the accelerometer. The value should be in the range \mbox{[}0, 4095\mbox{]}. The formula for calculating the inner sample rate is\+: 1.\+125k\+Hz / sample\+\_\+rate\+\_\+div
\item \doxylink{namespaceicm20948_a270e710b092b62b0a289753093c2a673}{accel\+\_\+scale} {\ttfamily scale}\+: The full-\/scale range of the accelerometer measurements. This defines the maximum acceleration (in g) that the sensor can measure. The available options are defined in the {\ttfamily \doxylink{namespaceicm20948_a270e710b092b62b0a289753093c2a673}{accel\+\_\+scale}} enumeration.
\item bool {\ttfamily dlpf\+\_\+enable}\+: A flag indicating whether the Digital Low-\/\+Pass Filter (DLPF) is enabled. If true, the DLPF is active and the cutoff frequency is determined by {\ttfamily dlpf\+\_\+config}.
\item \doxylink{namespaceicm20948_a11320e5ffc65dc1997d850a2747b1857}{accel\+\_\+dlpf\+\_\+config} {\ttfamily dlpf\+\_\+config}\+: The configuration settings for the DLPF, determining the filter\textquotesingle{}s cutoff frequency. The available options are defined in the {\ttfamily \doxylink{namespaceicm20948_a11320e5ffc65dc1997d850a2747b1857}{accel\+\_\+dlpf\+\_\+config}} enumeration.
\end{DoxyItemize}
